# Plex to Cache - Unraid Plugin

[DE] Ein Unraid Plugin, das Medien automatisch vom Array auf den Cache verschiebt, sobald ein Stream startet.
[EN] An Unraid plugin that automatically moves media from the array to the cache when a stream starts.

1. Go to the **Plugins** tab in your Unraid web interface.
2. Click on **Install Plugin**.
3. Copy the following link and paste it into the field:
   `https://raw.githubusercontent.com/MajorPain007/unraid-move-to-cache/main/plex_to_cache.plg`
